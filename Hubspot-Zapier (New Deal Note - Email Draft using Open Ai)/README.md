# Zapier Workflow: Auto-Draft Email on HubSpot Note Creation

## 📌 Project Overview
This Zapier automation triggers whenever a new note is added to a HubSpot deal. It seamlessly processes the note content and generates a draft email in Gmail, reducing the need for manual follow-up emails and improving communication efficiency.

## 🔧 Key Features & Functionality

### 🔄 HubSpot Integration
- Monitors new note creation on HubSpot deals.
- Retrieves related deal and owner details for context.

### 🧠 AI-Powered Message Generation
- Uses OpenAI's ChatGPT to process the note content and generate professional email responses.

### 🧾 Custom Output Formatting
- JavaScript code module formats the AI output into a polished, structured email body.

### 📬 Gmail Draft Creation
- Automatically creates a draft email in Gmail, ready for review and sending.

## 🛠️ Technology Stack
- **Zapier** – for automation and integration orchestration  
- **HubSpot** – as the CRM and note source  
- **Gmail** – for drafting follow-up emails  
- **OpenAI (ChatGPT)** – for generating email content  
- **JavaScript (Zapier Code module)** – for formatting and logic control

## 📈 Business Impact
- Minimizes time spent on manual email drafting  
- Ensures faster, more consistent communication with leads or clients  
- Provides a scalable solution for integrating AI into internal workflows
